Car Performance Visualization Dashboard

About This Dashboard

This interactive car performance dashboard is built using D3.js and presents insights from real-world automobile data. It showcases key metrics like fuel efficiency, horsepower, weight, and origin trends using dynamic and linked visualizations.

Click on any chart element to explore filtered views across all graphs. Use the reset buttons to return to full data.
